## ◈ CUM FUNCȚIONEAZĂ

La momentul potrivit (pre-lansare, update major, campanie), agentul:
1. Citește SPEC.md pentru a înțelege produsul
2. Citește 3MEK.md pentru stadiul firmei și analitici existente
3. Citește BUSINESS_LOGIC.md pentru audiență și strategie
4. Generează scripturile complete gata de copy-paste în tool-ul AI dorit
5. Oferă variante (A/B) pentru testare

## ◈ COPY MARKETING (Text pentru postări)

### Framework hook-uri eficiente

**Pentru TikTok/Reels (primele 3 secunde sunt critice):**
```
Hook tip 1 (Problem): 
"Ești [PROBLEMA_COMUNA]? 😤 Uite ce am găsit..."

Hook tip 2 (Revelație):
"Nu știam că există asta până ieri 👀"

Hook tip 3 (Contrast):
"Înainte: [SITUATIA_REA]. Acum: [SITUATIA_CU_APP]"

Hook tip 4 (Cifre):
"[TIMP_ECONOMISIT] cu [APP_NAME] 🤯"

Hook tip 5 (POV):
"POV: tocmai ai descoperit [APP_NAME]"
```

**Pentru Instagram caption:**
```
[HOOK din lista de mai sus]

[2-3 propoziții despre beneficiu — nu feature]

[Social proof: "Mii de oameni deja folosesc..."]

[CTA: "Link în bio" / "Descarcă gratuit" / "Încearcă acum"]

[3-5 hashtag-uri relevante]
```

**Pentru LinkedIn (B2B):**
```
[Observație despre industrie sau problemă]

Am construit [APP_NAME] pentru a rezolva asta.

Iată cum funcționează:
→ [Pas 1]
→ [Pas 2]  
→ [Rezultat]

[Rezultate sau testimoniale dacă există]

[CTA direct și profesional]

[Hashtag-uri: 3-5, specifice industriei]
```

## ◈ SCRIPTURI VIDEO (pentru CapCut / DaVinci)

### Script demo 30 secunde:
```
[0-3s]  HOOK VIZUAL: [ce apare pe ecran, text overlay]
[3-8s]  PROBLEMA: [situația pe care o rezolvă app-ul]  
[8-20s] SOLUȚIE: [demo al feature-ului principal, cu ecran înregistrat]
[20-27s] REZULTAT: [ce câștigă userul]
[27-30s] CTA: "[Descarcă gratuit — link în bio]"

Muzică: [tip de muzică — energică/calmă/upbeat]
Voiceover: [da/nu — dacă da, textul exact]
Text overlay: [ce scrie pe ecran la fiecare moment]
```

### Script tutorial 60 secunde:
```
[0-5s]   Hook: "Iată cum să [REZULTAT DORIT] în mai puțin de 2 minute"
[5-15s]  Setup: "Descarcă [APP] și creează-ți cont în 30 de secunde"
[15-45s] Tutorial pas cu pas: [pașii exacti în app, cu demo screen recording]
[45-55s] Rezultat: "Uite ce am obținut — [REZULTAT]"
[55-60s] CTA: "Descarcă gratuit — link în bio. Salvează videoul!"
```

## ◈ MATERIALE APP STORE

### Titlu și subtitlu optimizat (generat de agent):
```
Titlu (max 30 chars):    [KEYWORD PRINCIPAL] — [BENEFICIU SCURT]
Subtitlu (max 30 chars): [AL DOILEA KEYWORD + BENEFICIU]
```

### Descriere scurtă (80 chars pentru Google Play):
```
[BENEFICIU PRINCIPAL] cu [FEATURE CHEIE]. [DIFERENTIATORUL] 🚀
```

### Cuvinte cheie (max 100 chars, fără spații):
```
[keyword1],[keyword2],[keyword3],[keyword4],[keyword5]
```

### Descriere lungă (primele 255 chars sunt critice):
```
[HOOK] — [ce face app-ul în mod simplu]

✅ [Feature 1] — [beneficiu]
✅ [Feature 2] — [beneficiu]  
✅ [Feature 3] — [beneficiu]

[Social proof dacă există]

[Cum funcționează — 3 pași simpli]

[CTA final]
```
