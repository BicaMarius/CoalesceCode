🔗 GitHub Auto-fetch

Introdu URL-ul repository-ului tău GitHub și folderul cu blueprints. Agentul va extrage automat toate METADATA_JSON din fișierele Blueprint 02.

Format URL: https://github.com/USER/REPO
Folder blueprints: docs/pushes (default)

⚠️ Repository-ul trebuie să fie public sau să ai un Personal Access Token.

Importă date proiect

Lipește JSON-ul generat din blueprinturile proiectului. Fiecare blueprint conține un bloc METADATA_JSON — extrage valorile și construiește obiectul de mai jos.

Format așteptat: {"project":"Nume","description":"...","repos":{"repo-name":{"color":"#hex","label":"Label","short":"XX"}},"events":[...]}

Tipuri de events: "kickoff" · "push" · "mvp"
